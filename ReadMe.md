# VIDEO PLAYER
The main idea of this project is to make a video player work with controls.

# Main Functionality
- Play and Pause Video
- Increase to Fullscreen 
- Increase and Reduce the Volume
- The Volume Slider..
- The Video Progress Bar Seek.
- Increase and reduce the playback rate.
- Currenttime of the video changes when its being played
- Muting and Unmuting the  Video with the Icon changes.
